# 法师

## 1. 创建法师账户

```
xchain-cli account new --account 1234567890400001 --fee 1000
xchain-cli transfer --to XC1234567890400001@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890400001@xuper

```

### 2. 申请成为法师

```bash
xchain-cli wasm invoke memorial_hall --method apply_log -a '{"creditcode":"fashi123456", "proof":"xxxxx"}' --account XC1234567890400001@xuper
xchain-cli wasm invoke memorial_hall --method approve_log -a '{"id":"XC1234567890400001@xuper"}' --account XC1234567890100001@xuper
xchain-cli wasm invoke memorial_hall --method is_log --account XC1234567890400001@xuper
xchain-cli wasm invoke memorial_hall --method list_log --account XC1234567890100001@xuper

xchain-cli wasm invoke memorial_hall --method recusal_log -a '{"id":"XC1234567890400001@xuper"}' --account XC1234567890100001@xuper
```
