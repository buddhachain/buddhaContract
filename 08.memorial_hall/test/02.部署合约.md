# 部署

## 1. 创建合约部署者 1234567890100000

```bash
xchain-cli account new --account 1234567890100000 --fee 1000
xchain-cli transfer --to XC1234567890100000@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890100000@xuper

```

## 2. 部署

```bash
xchain-cli wasm deploy --account XC1234567890100000@xuper --cname memorial_hall --fee 5000000 --runtime c ../buddhaContract/08.memorial_hall/08.memorial_hall.wasm

```

## 3. 升级

```bash
xchain-cli wasm upgrade --account XC1234567890100000@xuper --cname memorial_hall --fee 5000000 ../buddhaContract/08.memorial_hall/08.memorial_hall.wasm

xchain-cli wasm invoke --fee 1000000 memorial_hall --method is_deployer --account XC1234567890100000@xuper

```
