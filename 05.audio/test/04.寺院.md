# 寺院

## 1. 创建寺院账户

```
xchain-cli account new --account 1234567890300001 --fee 1000
xchain-cli transfer --to XC1234567890300001@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890300001@xuper
```

## 2. 申请成为寺院

```bash
xchain-cli wasm invoke audio --method publish -a '{"unit":"佛山顶观音寺", "creditcode":"F110300007", "address":"浙江丽水佛顶山", "proof":"xxxxx"}' --account XC1234567890300001@xuper
xchain-cli wasm invoke audio --method approve_guaranty -a '{"id":"XC1234567890300001@xuper"}' --account XC1234567890200001@xuper
xchain-cli wasm invoke audio --method is_guaranty --account XC1234567890300001@xuper
xchain-cli wasm invoke audio --method list_guaranty --account XC1234567890200001@xuper

xchain-cli wasm invoke audio --method recusal_guaranty -a '{"id":"XC1234567890300001@xuper"}' --account XC1234567890200001@xuper
```
