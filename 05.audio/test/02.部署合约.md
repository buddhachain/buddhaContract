# 部署

## 1. 创建合约部署者 1234567890000000

```bash
xchain-cli account new --account 1234567890000000 --fee 1000
xchain-cli transfer --to XC1234567890100000@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890100000@xuper

```

## 2. 部署

```bash
xchain-cli wasm deploy --account XC1234567890100000@xuper --cname audio --fee 5000000 --runtime c ../buddhaContract/05.audio/05.audio.wasm

```

## 3. 升级

```bash
xchain-cli wasm upgrade --account XC1234567890100000@xuper --cname audio --fee 5000000 ../buddhaContract/05.audio/05.audio.wasm

xchain-cli wasm invoke --fee 1000000 audio --method is_deployer --account XC1234567890100000@xuper

```
