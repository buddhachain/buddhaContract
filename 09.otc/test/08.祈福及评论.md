# 善举祈福及点评流程

## 1. 创建普通用户

```bash
xchain-cli account new --account 1234567890500001 --fee 1000
xchain-cli transfer --to XC1234567890500001@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890500001@xuper

```

## 2. 添加删除修改点评标签

```bash
xchain-cli wasm invoke --fee 1000000 otc --method add_commentlabel -a '{"id":"1", "desc":"处理及时"}' --account XC1234567890100001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method delete_commentlabel -a '{"id":"1"}' --account XC1234567890100001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method update_commentlabel -a '{"id":"1", "desc":"处理及时"}' --account XC1234567890100001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method find_commentlabel -a '{"id":"1"}' --account XC1234567890500001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method list_commentlabel --account XC1234567890100001@xuper

```

## 3. 添加删除修改祈求善举前点评

```bash
xchain-cli wasm invoke --fee 1000000 otc --method add_beforecomment -a '{"kdid":"1", "satisfaction":"0", "labels":"[\"1\"]","comment":"够前评论，不孬","timestamp":"xxxxxx"}' --account XC1234567890500001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method delete_beforecomment -a '{"kdid":"1"}' --account XC1234567890500001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method update_beforecomment -a '{"kdid":"1", "satisfaction":"0", "labels":"[\"1\"]","comment":"够前评论，不孬","timestamp":"xxxxxx"}' --account XC1234567890500001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method find_beforecomment -a '{"kdid":"1","owner":"XC1234567890500001@xuper"}' --account XC1234567890100001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method find_beforecomment -a '{"kdid":"1","owner":"XC1234567890500001@xuper"}' --account XC1234567890400001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method find_beforecomment -a '{"kdid":"1"}' --account XC1234567890500001@xuper

xchain-cli wasm invoke --fee 1000000 otc --method list_beforecomment -a '{"kdid":"1"}' --account XC1234567890100001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method list_beforecomment -a '{"kdid":"1"}' --account XC1234567890400001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method list_beforecomment -a '{"kdid":"1"}' --account XC1234567890500001@xuper

```

## 4. 用户祈求善举

```bash
xchain-cli account balance XC1234567890500001@xuper

xchain-cli wasm invoke --fee 1000000 otc --method pray_kinddeed -a '{"id": "1","kinddeed":"1","spec":"1","count":"2","timestamp":"11111"}' --account XC1234567890500001@xuper --amount 20
xchain-cli wasm invoke --fee 1000000 otc --method delete_pray_kinddeed -a '{"id": "1"}' --account XC1234567890500001@xuper 
xchain-cli wasm invoke --fee 1000000 otc --method find_pray_kinddeed -a '{"id": "1"}' --account XC1234567890500001@xuper 
xchain-cli wasm invoke --fee 1000000 otc --method list_pray_kinddeed --account XC1234567890100001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method is_user --account XC1234567890500001@xuper

```

## 5. 基金会成员授权法师或寺院上传的善举凭证

```bash
xchain-cli account balance XC1234567890400001@xuper

xchain-cli wasm invoke --fee 1000000 otc --method upload_kinddeedproof -a '{"orderid":"1", "sellid":"xxxxx", "timestamp":"xxxxxx"}' --account XC1234567890400001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method approve_kinddeedproof -a '{"orderid":"1"}' --account XC1234567890100001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method refuse_kinddeedproof -a '{"orderid":"1"}' --account XC1234567890100001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method find_kinddeedproof -a '{"orderid":"1"}' --account XC1234567890400001@xuper

xchain-cli wasm invoke --fee 1000000 otc --method list_kinddeedproof --account XC1234567890100001@xuper

```
