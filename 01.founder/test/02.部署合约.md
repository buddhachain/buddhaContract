# 部署

## 1. 创建合约部署者 1234567890000000

```bash
xchain-cli account new --account 1234567890000000 --fee 1000
xchain-cli transfer --to XC1234567890100000@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890100000@xuper

```

## 2. 部署

```bash
xchain-cli wasm deploy --cname founder --account XC1234567890100000@xuper --runtime c ../buddhaContract/01.founder/01.founder.wasm  --fee 5000000

```

## 3. 升级

```bash
xchain-cli wasm upgrade --cname founder --account XC1234567890100000@xuper ../buddhaContract/01.founder/01.founder.wasm --fee 5000000

xchain-cli wasm invoke founder --account XC1234567890100000@xuper --method is_deployer --fee 1000000

```
