# 申请成为基金会成员及提案

## 1. 创建基金会成员账户

```bash

# 创建基金会成员 1234567890000001
xchain-cli account new --account 1234567890000001 --fee 1000
xchain-cli transfer --to XC1234567890000001@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890000001@xuper

# 创建寺院 1234567890000002
xchain-cli account new --account 1234567890000002 --fee 1000
xchain-cli transfer --to XC1234567890000002@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890000002@xuper

# 创建法师 1234567890000003
xchain-cli account new --account 1234567890000003 --fee 1000
xchain-cli transfer --to XC1234567890000003@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890000003@xuper

```

## 2. 申请成为基金会成员

```bash
xchain-cli wasm invoke --fee 1000000 buddha --method apply_founder -a '{"desc":"我很好1"，"address":"我的老家在东北","timestamp":"2021-03-11 10:00:00"}' --account XC1234567890000001@xuper --amount 100000
xchain-cli wasm invoke --fee 1000000 buddha --method apply_founder -a '{"desc":"我很好2"，"address":"我的老家在东北","timestamp":"2021-03-11 10:00:01"}' --account XC1234567890000002@xuper --amount 100000
xchain-cli wasm invoke --fee 1000000 buddha --method apply_founder -a '{"desc":"我很好3"，"address":"我的老家在东北","timestamp":"2021-03-11 10:00:02"}' --account XC1234567890000003@xuper --amount 100000
xchain-cli wasm invoke --fee 1000000 buddha --method approve_founder -a '{"id":"XC1234567890000001@xuper"}' --account XC1234567890111111@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method approve_founder -a '{"id":"XC1234567890000002@xuper"}' --account XC1234567890111111@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method approve_founder -a '{"id":"XC1234567890000003@xuper"}' --account XC1234567890111111@xuper

xchain-cli wasm invoke --fee 1000000 buddha --method is_founder --account XC1234567890000001@xuper

xchain-cli wasm invoke --fee 1000000 buddha --method list_founder --account XC1234567890111111@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method list_founder --account XC1234567890000003@xuper

xchain-cli wasm invoke --fee 1000000 buddha --method recusal_founder -a '{"id":"XC1234567890000001@xuper"}' --account XC1234567890111111@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method recusal_founder -a '{"id":"XC1234567890000002@xuper"}' --account XC1234567890111111@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method recusal_founder -a '{"id":"XC1234567890000003@xuper"}' --account XC1234567890111111@xuper

```


## 3. 创建提案

```bash
xchain-cli wasm invoke --fee 1000000 buddha --method make_proposal -a '{"key":"burn_rate", "value":"10"， "timestamp": "xxxxxx" }' --account XC1234567890000001@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method delete_proposal -a '{"key":"burn_rate"}' --account XC1234567890000001@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method update_proposal -a '{"key":"burn_rate", "value":"15"}' --account XC1234567890000001@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method find_proposal -a '{"key":"burn_rate"}' --account 1234567890000001@xuper
xchain-cli wasm invoke --fee 1000000 buddha --method list_proposal --account XC1234567890000001@xuper

```

## 4. 各个基金会成员授权提案
