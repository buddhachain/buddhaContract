# 寺院

## 1. 创建寺院账户

```
xchain-cli account new --account 1234567890500001 --fee 1000
xchain-cli transfer --to XC1234567890500001@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890500001@xuper
```

## 2. 申请成为寺院

```bash
xchain-cli wasm invoke identity --account XC1234567890500001@xuper --method publish -a '{"unit":"佛山顶观音寺", "creditcode":"F110300007", "address":"浙江丽水佛顶山", "proof":"xxxxx"}' --fee 1000000
xchain-cli wasm invoke identity --account XC1234567890100001@xuper --method approve_guaranty -a '{"id":"XC1234567890500001@xuper"}' --fee 1000000
xchain-cli wasm invoke identity --account XC1234567890500001@xuper --method is_guaranty --fee 1000000
xchain-cli wasm invoke identity --account XC1234567890100001@xuper --method list_guaranty --fee 1000000

xchain-cli wasm invoke identity --account XC1234567890100001@xuper --method recusal_guaranty -a '{"id":"XC1234567890500001@xuper"}' --fee 1000000
```
