# 认证账户

## 1. 创建认证账户账户

```
xchain-cli account new --account 1234567890400001 --fee 1000
xchain-cli transfer --to XC1234567890400001@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890400001@xuper

```

### 2. 申请成为认证账户

```bash
xchain-cli wasm invoke identity --account XC1234567890400001@xuper --method apply_identifyuser -a '{"creditcode":"fashi123456", "proof":"xxxxx"}' --fee 1000000
xchain-cli wasm invoke identity --account XC1234567890200001@xuper --method approve_identifyuser -a '{"id":"XC1234567890400001@xuper"}' --fee 1000000
xchain-cli wasm invoke identity --account XC1234567890400001@xuper --method is_identifyuser --fee 1000000
xchain-cli wasm invoke identity --account XC1234567890200001@xuper --method list_identifyuser --fee 1000000

xchain-cli wasm invoke identity --account XC1234567890200001@xuper --method recusal_identifyuser -a '{"id":"XC1234567890400001@xuper"}' --fee 1000000
```
