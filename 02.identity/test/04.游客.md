# 游客

## 1. 创建游客账户

```
xchain-cli account new --account 1234567890200001 --fee 1000
xchain-cli transfer --to XC1234567890200001@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890200001@xuper

```

### 2. 操作

```bash
xchain-cli wasm invoke --fee 1000000 identity --method add_visitor -a '{"nickname":"visitor.nickname", "wechat":"visitor.wechat"}' --account XC1234567890200001@xuper
xchain-cli wasm invoke --fee 1000000 identity --method is_visitor --account XC1234567890200001@xuper
xchain-cli wasm invoke --fee 1000000 identity --method list_visitor --account XC1234567890200001@xuper

```
