# 用户

## 1. 创建用户账户

```
xchain-cli account new --account 1234567890300001 --fee 1000
xchain-cli transfer --to XC1234567890300001@xuper --amount 10000000000000 --keys data/keys
xchain-cli account balance XC1234567890300001@xuper

```

### 2. 申请成为用户

```bash
xchain-cli wasm invoke --fee 1000000 otc --method apply_log -a '{"creditcode":"fashi123456", "proof":"xxxxx"}' --account XC1234567890300001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method approve_log -a '{"id":"XC1234567890300001@xuper"}' --account XC1234567890200001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method is_log --account XC1234567890300001@xuper
xchain-cli wasm invoke --fee 1000000 otc --method list_log --account XC1234567890200001@xuper

xchain-cli wasm invoke --fee 1000000 otc --method recusal_log -a '{"id":"XC1234567890300001@xuper"}' --account XC1234567890200001@xuper
```
